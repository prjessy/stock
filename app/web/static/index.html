⚙️ 설정

📡 서버 알림 종류 (복수 선택 · 폰 꺼도 수신)
선택한 알림을 서버가 장중(평일 08~20시) 자동 감시해 텔레그램+카카오로 보냅니다 — 브라우저를 안 켜도 수신됩니다. 더듬이1·4 발송은 곧 추가됩니다.
등락률(증감) 단계 · 횟수
±% × 최대 회
단계의 배수마다 알림. 증감 알림을 켜면 서버가 텔레그램·카카오로 발송(폰 꺼도 수신). 카드 🔔 강조도 같은 기준.
오늘 흐름 간격
분
'🕔 오늘 흐름' 표 간격(1~120분). 1분봉을 받아 합쳐 표시하므로 아무 분이나 가능합니다. 값은 이 브라우저(localStorage)에 저장됩니다.
실시간 묶음
초
'⚡실시간' 라인을 N초 간격으로 묶어 표시합니다(1=원본 초단위). 초 단위는 페이지를 연 시점부터 쌓인 데이터에만 적용됩니다. 값은 이 브라우저(localStorage)에 저장됩니다.
🔔 알림 방식 (복수 선택)
목표가 도달 시 선택한 방식으로 알립니다(복수 가능, localStorage 저장). 자동 더듬이 사이렌은 별도로 항상 텔레그램으로 발송됩니다.

🎯 목표가 알림
종목별 목표가 도달 시 카드 강조 + 브라우저 알림 + 소리. 목표가가 현재가보다 높으면 '이상', 낮으면 '이하' 도달 시 발동합니다. 한 번 발동하면 값을 다시 설정할 때까지 반복하지 않습니다. (대시보드가 열려 있을 때 동작 · localStorage 저장)
