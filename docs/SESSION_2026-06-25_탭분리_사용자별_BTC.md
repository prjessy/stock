## 🗺️ 향후 로드맵
### 상용화 A안 (정보·분석 앱 — 권장 1단계)
- **PWA → Play Store(TWA)** 출시: manifest+service worker 추가, assetlinks, 개인정보처리방침/약관/투자위험 고지.
- **카카오 로그인 추가**(구글 병행) — 한국 사용자 가입률↑.
- **데이터 라이선스**: KIS 실시간 대중 재배포는 약관 이슈 → 공개는 지연/무료소스, 실시간은 셀프호스트/유료.

### 확장 (동접 늘면)
- SQLite **WAL 모드**(즉효) → 동시접속 많아지면 **PostgreSQL**.
- 폴링(1초) → **푸시(SSE)** / 폴링 간격↑.
- 폴러를 **별도 프로세스**로 분리 후 웹 멀티워커.
- 진짜 천장 = **단일 KIS키 20req/s + 사용자별 종목 합집합** → 대중 공개 시 데이터 소스 전략 필요.

### B안 (자동매매 다사용자 — 후순위)
- 사용자별 KIS 앱키 암호화 저장 + 계좌 격리(KIS 소비자 위임 OAuth 없음 → 각자 앱키 입력).
- **법규**: 타인 계좌 자동매매=투자일임 인가 필요 → 법률검토 필수. "정보·도구 제공" 포지셔닝으로 회피.
