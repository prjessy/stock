# 세션 정리 — 2026-06-23 · 구글 로그인 + 사용자별 매매일지 + 대시보드 재구성

## 한 일 (전부 완료 · 라이브 작동 중 `https://jessystock.com`)

### 1. 구글 로그인(OAuth)
- `app/auth/google_auth.py` (urllib, 카카오와 동일 무의존 방식)
- 라우트(`app/web/api.py`): `/api/auth/google/login` · `/callback` · `/api/auth/logout` · `/api/auth/me`
- 세션 = 랜덤 `sid` httponly+secure 쿠키 ↔ DB `sessions` 테이블(30일)
- 키는 VPS `.env` 에만(`GOOGLE_CLIENT_ID/SECRET/REDIRECT_URI`) — 커밋 금지, 미설정 시 로그인 버튼 자동 숨김
- **VPS 배포 + 키 입력 완료** → `/api/auth/me` = `{configured:true, mode:journal}` 확인

### 2. 사용자별 매매일지 (📔 탭)
- DB `users` / `journal` 테이블(`app/storage/models.py`) + Repository CRUD(`app/storage/db.py`)
- API: `GET/POST /api/journal`, `PUT/DELETE /api/journal/{id}` — **본인 소유만(user_id 강제)**, 비로그인 401
- 입력 항목: 날짜 · 구분(매수/매도/메모) · 종목명 · 종목코드 · 가격 · 수량 · 이유/근거 · 복기/메모
- 프론트: 비로그인 게이트 → 로그인 시 입력폼 + 목록표 + 삭제

### 3. 인증 3모드 토글 (⚙️ 설정)
- `off`(미사용·로그인/일지탭 숨김) / `journal`(매매일지만 로그인) / `full`(사이트 전체 로그인)
- `GET/POST /api/auth-config` → `data/auth_config.json`(gitignore), 변경엔 **TRADE_PASSWORD** 필요
- `full` 모드: 비로그인 시 클라이언트 전체화면 게이트
- **현재 모드 = journal**

### 4. UI 재구성
- **대시보드(써머리)**: 외국인/기관/개인 수급 카드 추가(`/api/investor`)
- **💼 포트폴리오 탭 신설**: 총자산 히어로 + 🍩 종목별 비중 도넛 + 보유종목 표(`/api/balance`) — 써머리에 있던 총자산/비중을 여기로 이동
- **차트 탭**: 삼성·하이닉스 종가는 기존 캔들(종가 포함) + 일별 종가 표로 유지

### 5. 기타
- 웹앱 startup 에 `init_db()` 호출 추가 → VPS DB 에 새 테이블 자동 생성(idempotent)
- `app/config.py`(GOOGLE_*, REQUIRE_LOGIN), `.env.example`, `docs/GOOGLE_LOGIN_SETUP.md`

## 배포/검증
- GitHub push (main 직접) → VPS `git pull && systemctl restart stock-watchdog`
- 검증: 공개 도메인 `configured:true`, `/api/auth/google/login` → 구글 동의화면 정상 리디렉션
- 시크릿 안전: `.env`/`key*.txt` 미추적, 커밋에 시크릿 없음

## ⚠️ 남은 체크 (선택)
- **다른 사용자도 로그인하려면**: Google Cloud → OAuth 동의 화면 → **대상(Audience)** → 게시 상태 **'프로덕션(In production)'** 으로 전환. '테스트' 상태면 등록된 테스터(오너)만 로그인됨.
