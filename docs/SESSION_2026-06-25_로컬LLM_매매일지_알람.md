## ✅ 완료 (라이브 배포·검증됨)

### 1. AI 백엔드 분리 + 로컬 LLM
- AI 분석 8곳(더듬이2/3·매수판단·브리핑·EOD·보유케어·ETF수급·마케팅·주간복기)을 공통 어댑터 `app/llm.py`로 통합.
- `.env LLM_PROVIDER` 토글: `anthropic`=Claude(haiku) / `openai`=사설 OpenAI 호환 로컬 LLM.
- **현재 구성: 앱=Claude(haiku), 텔레그램 봇(Hermes)=로컬 LLM(qwen3.5-122b).**
- **과금 원인 규명**: Anthropic 크레딧 소진(앱+Hermes 둘 다 Claude 사용 중이었음). 사용자가 재충전.
- Hermes 게이트웨이도 `config.yaml` model→로컬(openai/qwen)로 전환(백업 후, 안정 확인).

### 2. 매매일지 대폭 확장
- 입력: **배당** 구분, **세금(원)**, **통화 KRW/USD + 환율**, 총 거래대금 실시간 미리보기.
- **실현손익 자동계산**(매도 시 가중평균 평단 대비), **투자 캘린더**(월간+날짜별 현금흐름), **월간 리포트**(요약+종목별+인쇄/PDF).
- **공모주/일반주 구분**(폼 선택 + 목록 배지 + 전체/일반주/공모주 필터).
- DB 컬럼은 기존 데이터 보존 마이그레이션(`storage/models.py`·`db.py`).

### 3. 수급 카드
- **PER·RSI(14)** 추가(차트탭·대시보드). 단일 `refreshSupply()`(60초)로 통합 → "대시보드만 안 뜨던" 버그 수정.

### 4. 자동매매
- **저장 누락 버그 수정**: ON 토글·수량만 바꿔도 저장됨(이전엔 규칙값 없으면 종목 드롭).
- 「💾 저장·적용」 버튼(전체 ON 토글 옆에도 추가) + 비번 미입력 시 입력창 + "장중 09:00~15:30 자동반영" 안내 + 조건없는 ON 경고.
- 미국 종목 **가격 전용 카드**(KIS 국내 주문만 지원 → 미국 자동매매 미지원).

### 5. 동적 종목 한글명 해석
- 새로 추가한 국내 종목이 이름 대신 코드로 표시되던 문제(예: 279570) 수정.
- KIS `inquire-price`엔 이름이 없어 → **주식기본조회(CTPF1604R) `search-stock-info`**로 한글명 조회(KRX 전종목 리스트가 차단된 해외 VPS 대비), 프로세스 캐시. `kis_price.get_name`.

### 6. 대시보드 BTC
- 금·환율 카드에 **₿ 비트코인(USD)** 추가(`fintech._MARKET_DEFS` + 대시보드 필터에 btc 포함). (값=FDR 기준, 지연 가능)

### 7. 아이폰 구글 로그인 + 인증
- OAuth `state`를 프로세스 메모리 → **파일 보관**(`data/oauth_states.json`, 앱 재시작·재배포에도 생존)+TTL 1시간 → `STATE MISMATCH` 수정.
- 임시 인증 우회(bypass) **OFF** → 각자 구글 로그인=**개인별 매매일지 분리**.
- 로그인 콜백(`/?login=ok`) 직후 **매매일지 탭 자동 전환**(대시보드 대신).

### 8. 버전 표시 + 릴리즈 노트
- 푸터에 **v1.2.0** + 📋 펼침 릴리즈 노트, `RELEASE_NOTES.md` 갱신.

### 9. 지인 자동 알람 (멀티 수신 + 개인정보 분리)
- `notify_all(subject, msg, shared=False)`에 `shared` 플래그.
  - `shared=True`(시황: 등락🔔·더듬이1🟣·더듬이4🟢·장마감📊·더듬이2/3 사이렌🚨) → 지인 목록 전체로 팬아웃.
  - `shared=False`(개인정보성: 목표가🎯·보유케어💼·주간복기🗓️·자동매매 체결🤖) → 본인(홈채널)만.
- `.env ALERT_TELEGRAM_IDS`는 **`id:이름` 형식**(헷갈림 방지). 발송은 id만, 이름은 표시용.
- 앱 🔔 알람 탭에 **지인 추가 가이드** + **현재 수신자 이름 목록**(`/api/alert-recipients`).
- 현재 시황 알람 수신 4명 등록·전송 테스트 OK.

## 🔧 운영 메모
- 배포: **git 커밋 금지**, 변경 파일 scp 직접 전송 후 `systemctl restart stock-watchdog`(파이썬) / 정적(html)은 새로고침.
- 앱 AI 백엔드 전환: `.env LLM_PROVIDER` = `anthropic`(현재) / `openai`(로컬). 로컬 전환 시 `LLM_*` 필요.
- 알람 수신자: `.env ALERT_TELEGRAM_IDS=id:이름,...` 편집 후 재시작.
- Hermes 봇 모델/수신자: VPS의 별도 CLI(`hermes config`·`hermes pairing`).
- 접속/식별 상세는 로컬 메모리(`local-llm-migration`, `alert-system-state`, `google-auth-journal`, `stock-watchdog-deploy`) 참조.
