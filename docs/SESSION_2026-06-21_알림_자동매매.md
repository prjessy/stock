## ✅ 완료 (라이브)

### 1. 시세 표시
- **정규장(KRX) + NXT 애프터마켓 병기** + 기준 명시. (증권사·네이버의 "시간외 350,500"은 NXT 값이었음 — KIS 시장코드 `NX`로 조회). 카드에 `KRX 정규장 / NXT 애프터` 두 줄.
- **레버리지 ETF는 NXT 제외**(시간외 없음 → 0원 안 뜸).
- **한국 종목도 휴장 시 `🕘 휴장·종가(stale)` 라벨**(기존엔 미국만). `registry.quote` + `core.market.is_market_open`.

### 2. 미국 브리핑(🇺🇸 미국 시장 동향)
- **직전 정규장 기준일 인식**(주말·미 공휴일 건너뜀, `last_us_trading_day`). "밤사이" 표현 제거 → 휴장인데 "혼조세"라던 오류 수정.
- **찰떡 써머리 파리티**: 변경/신규 배지(status) + 핵심어·퍼센트 하이라이트. **동기화 6시 유지**(장 개장 전).

### 3. 알림 서버화 1단계 (폰 꺼도 수신) — **사용자 폰 수신 검증됨**
- 설정을 **서버 저장**(`data/alert_config.json`), API `GET/POST /api/alert-config`. ⚙️설정에 4종 멀티선택.
- **증감(±%)**: 단계(step) 배수마다 × 최대 횟수(count). 예 **3%×3회 → ±3·6·9%**. 실시간 미리보기.
- **목표금액**: 도달 시.
- `app/core/alert_watch.py`(30초, 장중·非stale) → `notify_all`(텔레그램 Hermes + 카카오 동시). 클라는 팝업/소리만(중복 제거).

### 4. 자동매매 1단계 — 주문 모듈 + 수동 테스트 버튼
- `app/trading/kis_order.py` `OrderClient`: 현금주문(매수 TTTC0802U/매도 TTTC0801U), hashkey 서명, 토큰 재사용.
- **지정가 전용**(시장가 차단), **1주 하드캡**(`trade_max_qty`), 계좌 미설정 거부.
- API `POST /api/order`(지정가 필수), `GET /api/order/status`. UI ⚙️설정 '🧪 주문 테스트'(지정가 입력+현재가 자동채움+확인창).
- VPS `.env`에 **KIS_CANO 이미 설정**(`실전`, max_qty 1). 결과 카톡·텔레그램 통지.

### 5. 기타
- 보안: `.env`/`key*.txt` git 안전 검증 완료, `docs/SECURITY_ENV.md`.
- `docs/NOTIFY_STATUS.md`(알림 현황·계획).
- 더듬이4 KIS 호출 throttle 0.15초(초당 20 한계 여유).

## ⏳ 미완료 / 검증 대기

### 평일·장중 검증 필요 (오늘 일요일이라 불가)
- [ ] **증감·목표 알림 실발송** — 월요일 장중(±3·6·9% 또는 목표가 도달 시 폰 수신).
- [ ] **주문 테스트 실체결** — 월요일 장중(09:00~15:30). 오늘은 "장 미개장" 거부만. 안전팁: 지정가를 현재가와 멀게(매수=낮게/매도=높게) 넣으면 체결 없이 접수만 검증.
- [ ] **더듬이1·4 실동작** — 오전 9시 1회 발송. 특히 **더듬이4는 KODEX 200 구성종목이 주말엔 빈값** → 평일에 ① 구성종목이 채워지는지 ② 응답 필드명(`stck_shrn_iscd` 등)이 맞는지 검증 필요(`get_etf_constituents` 방어적 파싱).

### 구현 남음
- [ ] **자동매매 2단계 — 삼성전자 ±2% 자동 매수/매도**.
  - alert_watch가 이미 ±% 감지 → 거기에 `OrderClient` 연결.
  - **결정 필요**: -2% 매수 기준(전일종가 대비?), +2% 매도 기준(전일종가 대비? vs 매수가 대비 익절?).
  - `TRADE_ENABLED=true`(현재 false) + 안전장치: **1일 주문 한도·중복방지·kill-switch** 필수.
- [ ] 더듬이4 '신규 매수 유입' 기준 튜닝(현재: 당일 순매수>0 & 직전 5일합−당일≤0).

## 💡 검토 아이디어 — "발목(저점권) 감지" 알림/매수 트리거
완전 바닥은 불가하나 **저점권은 감지 가능**(이미 `feed.py`에 지표 보유). 여러 조건 동시 충족 시 "저점권 진입" 알림:
- RSI 과매도(≤30) · 스토캐스틱 과매도(≤20) · 볼린저 하단 터치
- 52주 저점/최근 저점/피보나치 지지선 근접
- **더듬이4 수급: 기관·외국인 매수세 신규 유입(바닥 매집 신호)**
→ 자동매매에서 단순 -2% 대신 **"저점권 + 수급유입"일 때 매수**로 고도화 가능. (지표 저점 ≠ 반등 보장, 보조용)
