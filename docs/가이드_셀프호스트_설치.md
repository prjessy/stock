# 셀프호스트 설치 가이드 — 내 계좌로 내 주식앱 돌리기

> **핵심 개념**: 이 앱은 **모든 설정·키를 `.env` 파일에서** 읽습니다. 그래서 **각자 자기 서버에 앱을 올리고 자기 `.env`(자기 KIS 계좌 키)를 넣으면**, 그 사람만의 독립 인스턴스가 됩니다.
> → 프로세스·DB·계좌가 통째로 분리되므로 **"계좌 격리"가 저절로 해결**됩니다.
> → 매매도 **본인이 본인 계좌를 본인이 운용**하는 것이라 법적 부담도 낮습니다(타인 자금 운용=투자일임 인가 이슈를 피함).

---

## 0. 누구나 가능한가? (솔직한 현실)

**기술적으로는 가능**합니다. 코드가 전부 `.env` 구동이고, `deploy/deploy.sh`가 설치를 자동화합니다.
다만 가이드로도 **못 없애는 진입장벽 3가지**가 있습니다:

| 장벽 | 설명 | 난이도 |
|---|---|---|
| ① KIS OpenAPI 신청 | 본인이 한국투자증권 계좌 + 개발자센터에서 OpenAPI 사용신청·승인 → 앱키 발급 | 누구나(시간 걸림) |
| ② 서버 | VPS(월 $4~6) 또는 **항상 켜 두는 PC** 필요. 폰 꺼도 자동매매하려면 24시간 서버 | 중 |
| ③ 터미널 | SSH 접속·`.env` 편집(복붙 수준이지만) | 컴퓨터에 익숙한 사람이면 OK |

➡️ **"컴퓨터/터미널에 어느 정도 익숙한 사람"이면 이 가이드로 충분히 가능**합니다.
완전 비개발자 대중까지 쉽게 하려면 → 도커 원클릭 이미지나 호스팅 대행이 필요(그건 다시 SaaS 영역).

## 1. 준비물 체크리스트

- [ ] **한국투자증권 계좌** + **OpenAPI 앱키/시크릿/계좌번호** (필수 — 2번 참고)
- [ ] **서버 1대**: 리눅스 VPS(권장) 또는 항상 켜 두는 PC(윈도우/맥)
- [ ] (선택) 도메인 + HTTPS — 없으면 `http://서버IP:8000` 으로 접속
- [ ] (선택) 구글 OAuth 키 — 매매일지 로그인용 (혼자 쓰면 없어도 됨)
- [ ] (선택) 텔레그램/카카오 — 알림용
- [ ] (선택) LLM(OpenAI 호환) 또는 Anthropic 키 — AI 분석/브리핑용

## 2. KIS OpenAPI 앱키 발급 (가장 중요 · 각자)

1. 한국투자증권 계좌 개설(이미 있으면 생략)
2. **KIS 개발자센터** https://apiportal.koreainvestment.com 접속 → 로그인
3. **OpenAPI 신청/이용신청** → 앱(App) 등록 → **APP KEY / APP SECRET** 발급
4. **계좌번호** 확인: 앞 8자리 = `KIS_CANO`, 상품코드 2자리(보통 `01`) = `KIS_ACNT_PRDT_CD`
5. **모의투자로 먼저 연습 권장**: 모의 도메인 `https://openapivts.koreainvestment.com:29443`, `KIS_PAPER=true`

> ⚠️ 앱키/시크릿은 **본인만 보관**. 절대 깃/카톡/캡처로 공유 금지(=계좌 탈취 위험).

## 3. 설치 — 리눅스 VPS (권장)

VPS(Ubuntu 22.04 등)에 SSH 접속 후:

```bash
# 1) 코드 받기
git clone https://github.com/prjessy/stock.git ~/stock-watchdog
cd ~/stock-watchdog

# 2) .env 만들기 (4번 템플릿을 보고 값 채우기)
cp .env.example .env
nano .env        # 저장: Ctrl+O, Enter / 종료: Ctrl+X

# 3) 자동 설치 + 상시가동 서비스 등록 (venv·패키지·systemd 자동)
bash deploy/deploy.sh
```

`deploy.sh`가 파이썬 가상환경·패키지 설치·systemd 서비스(`stock-watchdog`) 등록까지 합니다.
→ 부팅 시 자동 시작 + 죽으면 자동 재시작.

**코드 갱신 시(이후):**
```bash
cd ~/stock-watchdog && git pull && bash deploy/deploy.sh
```

**상태/로그 확인:**
```bash
systemctl status stock-watchdog
journalctl -u stock-watchdog -f      # 실시간 로그
```

접속: `http://서버IP:8000`  (도메인/HTTPS는 9번 참고)

## 3-b. 설치 — 윈도우/맥 PC (서버 없이 테스트)

> PC를 끄면 멈춥니다. 자동매매(폰 꺼도 동작)는 24시간 서버 필요.

```powershell
git clone https://github.com/prjessy/stock.git stock
cd stock
python -m venv .venv
.\.venv\Scripts\Activate.ps1        # 맥/리눅스: source .venv/bin/activate
pip install -r requirements.txt
Copy-Item .env.example .env          # 맥/리눅스: cp .env.example .env
notepad .env                         # 값 채우기
python -m app.web
```
접속: `http://localhost:8000`

## 4. `.env` 완전 템플릿 (복사해서 값만 채우기)

```ini
# ===== 필수: KIS 실시간 시세 =====
KIS_APP_KEY=여기에_앱키
KIS_APP_SECRET=여기에_시크릿
# 실전: https://openapi.koreainvestment.com:9443  /  모의: https://openapivts.koreainvestment.com:29443
KIS_DOMAIN=https://openapi.koreainvestment.com:9443

# ===== 감시 종목 (콤마구분 · 국내 6자리 / 미국 티커) =====
KR_SYMBOLS=005930,000660
US_SYMBOLS=NQ=F,^SOX

# ===== 주문/자동매매 계좌 (자동매매 쓸 때만) =====
KIS_CANO=계좌앞8자리
KIS_ACNT_PRDT_CD=01
KIS_PAPER=false            # true=모의투자(연습), false=실전
TRADE_ENABLED=false        # 주문 기능 마스터(기본 false=안전)
TRADE_MAX_QTY=1            # 1회 주문 최대 수량 하드캡(오발주 방지)
TRADE_PASSWORD=원하는비밀번호   # 비우면 모든 주문 거부(fail-closed). 자동매매 설정 변경에 필요

# ===== 저장소(SQLite 파일) =====
DB_PATH=data/stock.db

# ===== 시장 시간(KST) — 보통 그대로 =====
MARKET_OPEN=08:00
MARKET_CLOSE=20:00
THRESHOLDS=3.0,-3.0
POLL_INTERVAL_SECONDS=30

# ===== (선택) 구글 로그인 — 매매일지/사용자별 관심종목 =====
# 혼자 쓰면 비워도 됨(로그인 버튼 숨김). 여럿이 한 인스턴스를 쓸 때만 필요.
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URI=https://내도메인/api/auth/google/callback
REQUIRE_LOGIN=false

# ===== (선택) AI 분석/브리핑 — 둘 중 하나 =====
# A) OpenAI 호환 LLM(사설/로컬 포함)
LLM_PROVIDER=openai
LLM_ENDPOINT=
LLM_API_KEY=
LLM_MODEL=
# B) Anthropic Claude
ANTHROPIC_API_KEY=

# ===== (선택) 알림 =====
# 텔레그램: BotFather로 봇 생성 후 토큰/대상ID (docs/TELEGRAM_SETUP.md)
ALERT_TELEGRAM_IDS=
ALLOWED_CHAT_IDS=
# 카카오 '나에게 보내기' (docs/KAKAO_SETUP.md)
KAKAO_REST_API_KEY=
KAKAO_REDIRECT_URI=https://내도메인/api/kakao/callback

# ===== (선택) 부동산 실거래가(핀테크 탭) =====
MOLIT_API_KEY=
```

> **최소 구성**: `KIS_APP_KEY` / `KIS_APP_SECRET` 만 있어도 실시간 대시보드는 동작합니다.
> 자동매매까지 쓰려면 `KIS_CANO` + `TRADE_PASSWORD` 추가.

## 5. 자동매매 켜는 법 (주의)

1. 먼저 **`KIS_PAPER=true`(모의)** 로 충분히 연습 → 익숙해지면 실전 전환
2. `.env` 에 `KIS_CANO`, `TRADE_PASSWORD` 설정
3. 앱 → **🤖 자동매매 탭** → 종목별 규칙(매수%/기준가/손절/매도시각/수량) 설정 → **시간 범위**·**분할 스케줄** 옵션 → **전체 ON** 체크 → 비밀번호 입력 후 **적용**
4. 안전장치: 마스터 OFF가 기본, `TRADE_MAX_QTY`로 1회 수량 상한, 종목·행위당 하루 1회

> ⚠️ 실거래는 본인 책임. 처음엔 1주·모의로 검증하세요.

## 6. (선택) 외부 접속 — 도메인 + HTTPS

- 가장 쉬움: **Cloudflare Tunnel** (무료, 포트개방 불필요) — `deploy/CLOUDFLARE.md` 참고
- 대안: 도메인 없이 `http://서버IP:8000` 직접(단, 평문 — 로그인/주문엔 비권장)
- 구글 로그인 쓰려면 HTTPS 도메인 필수(쿠키 Secure)

## 7. 보안 수칙 (꼭)

- **`.env` 는 절대 git 커밋/공유 금지** (이미 `.gitignore` 처리됨). 권한 잠그기: `chmod 600 .env`
- `TRADE_PASSWORD` 는 길고 추측 어렵게. 코드/커밋메시지에 비번 넣지 말 것
- 실전 전환 전 **모의투자로 검증**
- `http://IP:8000` 직접 노출이 걱정되면 방화벽으로 8000 차단 + 터널만 사용

## 8. 트러블슈팅

| 증상 | 원인/해결 |
|---|---|
| 시세가 ~15분 지연/안 변함 | KIS 키 미설정 → 무료소스 폴백. `.env` 키 확인 후 재시작 |
| 토큰 발급 실패(403) | 앱키/시크릿 오타·줄바꿈. **scp/복붙 시 잘림 주의**. 같은 앱키를 두 곳에서 동시 사용 금지 |
| 변경했는데 그대로 | 브라우저 캐시(시크릿창/Ctrl+Shift+R) · 서버 `systemctl restart stock-watchdog` |
| 주문이 거부됨 | `TRADE_PASSWORD` 미설정(fail-closed) 또는 `KIS_CANO` 누락 |
| 로그인 안 풀림(아이폰) | HTTPS 도메인 + 구글 OAuth 리디렉트 URI 정확히 일치 확인 |
