주식 자동매매 봇을 만들어줘.

목표:
- 국내 또는 미국 주식 API와 연동 가능한 자동매매 봇
- 사람이 직접 매수/매도하지 않아도 조건에 따라 자동으로 거래
- 보수적인 리스크 관리 포함
- 추후 전략 교체가 쉬운 구조

필수 기능:
1. 장 시간 확인
2. 종목 스캔
3. 매수/매도 신호 생성
4. 자동 주문 실행
5. 손절/익절 관리
6. 포지션 크기 계산
7. 거래 로그 저장
8. 실시간 알림 전송
9. 백테스트 가능 구조
10. 에러 발생 시 재시도 및 예외 처리

원하는 구조:
- config
- broker api wrapper
- strategy
- risk manager
- trader
- notifier
- logger
- backtest

세부 요구사항:
- 전략은 교체 가능하게 인터페이스로 분리
- broker api는 추후 토스증권, 한국투자증권, 키움증권 등으로 바꿔 끼울 수 있게 설계
- 매수 전 현재 보유 종목, 주문 가능 금액, 일일 최대 손실 체크
- 주문 후 체결 여부 확인
- 손절, 익절, 트레일링 스탑 옵션 지원
- 모든 거래 기록을 파일로 저장
- 텔레그램 알림 지원
- .env 로 API 키 관리
- 예외 상황 발생 시 로그 남기고 안전 종료

개발 방식:
- 처음에는 mock broker 로 테스트 가능하게 작성
- 이후 실제 증권사 API adapter 만 추가하면 되게 설계
- 함수와 클래스 역할이 명확하도록 작성
- 유지보수하기 쉽게 파일 분리

추가로 해줄 것:
- 프로젝트 폴더 구조 제안
- requirements.txt 작성
- 실행 방법 README 작성

프로젝트명: AI 주식 자동매매 봇

목적:
시장 데이터를 기반으로 종목을 자동 탐색하고, 매매 조건 충족 시 자동으로 주문을 실행하며, 손절·익절·로그·알림까지 자동화하는 시스템 구축.

주요 기능:
1. 시장 데이터 수집
- 실시간 시세 조회
- 거래량, 변동성, 가격 변화율 수집
- 종목별 기본 정보 조회

2. 종목 스캔
- 거래량 상위 종목 추출
- 급등/급락 종목 탐색
- 사용자 정의 조건식 필터링

3. 전략 엔진
- 이동평균선 전략
- RSI 전략
- 볼린저밴드 전략
- 전략 교체 가능 구조

4. 주문 엔진
- 시장가 주문
- 지정가 주문
- 분할 매수
- 분할 매도
- 체결 여부 조회

5. 리스크 관리
- 손절 비율 설정
- 익절 비율 설정
- 트레일링 스탑
- 1회 최대 투자금 제한
- 일일 최대 손실 제한
- 동시 보유 종목 수 제한

6. 포트폴리오 관리
- 현재 보유 종목 조회
- 평균단가 계산
- 평가손익 계산
- 현금 잔고 확인

7. 로그 및 모니터링
- 주문 로그 저장
- 에러 로그 저장
- 전략 실행 로그 저장
- 성과 요약 리포트 생성

8. 알림
- 매수/매도 체결 알림
- 손절/익절 알림
- 에러 발생 알림
- 장 시작/종료 알림

9. 백테스트
- 과거 데이터 기반 전략 검증
- 승률, 손익비, MDD, 누적수익률 계산
- 전략별 비교 가능

10. 운영 안정성
- API 오류 재시도
- 네트워크 오류 예외 처리
- 장 종료 후 안전 종료
- 비정상 상태 발생 시 주문 중단
