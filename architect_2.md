아래 요구사항을 기준으로 기술 설계를 제안해라.

요구사항:
- 삼성전자 가격 조회
- 전일 종가 기준 +3%, -3% 알림
- Telegram 메시지 전송
- 중복 알림 방지
- 로컬 실행
- 나중에 Hermes notifier로 교체 가능해야 함
- .env 기반 비밀정보 관리

반드시 포함할 것:
- 파일 구조
- 각 파일의 책임
- 모듈 경계
- notifier 인터페이스 설계
- state 저장 방식
- 실행 흐름
- 에러 처리 방식
- 테스트 전략
- builder가 바로 구현할 수 있는 수준의 handoff

출력 형식:
1. 설계 요약
2. 파일 목록
3. 책임 분리
4. 함수/모듈 계약
5. 데이터 흐름
6. 예외 상황
7. 구현 순서
