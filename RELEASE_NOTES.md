## v1.2.0 — 2026-06-24

### AI 백엔드 분리 (비용 최적화)
- AI 분석 8곳(더듬이2/3·매수판단·브리핑·EOD·보유케어·ETF수급·마케팅·주간복기)을 공통 어댑터 `app/llm.py`로 통합.
- `.env` `LLM_PROVIDER` 토글: `anthropic`=Claude(haiku) / `openai`=사설 OpenAI 호환 로컬 LLM.
- **현재 구성**: 앱=Claude(haiku) · 텔레그램 봇(Hermes)=로컬 LLM(qwen3.5-122b).
- 토큰 사용량 기록 provider 무관화(로컬 모델 비용 0 집계).

### 매매일지 대폭 확장
- 입력: **배당** 구분, **세금(원)**, **통화 KRW/USD + 환율**, 총 거래대금 실시간 미리보기.
- **실현손익 자동계산**(매도 시 가중평균 평단 대비), **투자 캘린더**(월간+날짜별 현금흐름), **월간 리포트**(요약+종목별+인쇄/PDF).
- DB 컬럼은 기존 데이터 보존 마이그레이션.

### 수급 카드
- **PER·RSI(14)** 추가(차트탭·대시보드). 단일 `refreshSupply()`(60초)로 통합 → "대시보드만 안 뜨던" 버그 수정.

### 자동매매
- **저장 누락 버그 수정**: ON 토글·수량만 바꿔도 저장(이전엔 규칙값 없으면 종목 드롭).
- 「💾 저장 · 적용」 + 반영시점 안내 + 조건 없는 ON 경고. 미국 종목 **가격 전용 카드**(국내 주문만 지원).

### 아이폰 구글 로그인 수정
- OAuth `state`를 프로세스 메모리 → **파일 보관**(`data/oauth_states.json`, 재시작 생존), TTL 1시간.
- 원인: iOS Safari가 콜백에 쿠키 미전송(서버 state가 유일 검증) + 앱 재시작/만료로 state 소실 → `STATE MISMATCH`.

## v1.0.0 — 2026-06-20  🎉 첫 정식 릴리즈

"실시간 모니터링 + 알림" 대시보드의 첫 안정 버전.

### 핵심 기능
- **실시간 시세 대시보드** — 한국투자증권(KIS) OpenAPI로 국내 종목 **초단위** 조회. 브라우저 1초 폴링.
  - 워치리스트(5): 삼성전자·SK하이닉스·삼성전자레버리지(0193W0)·SK하이닉스레버리지(0193T0)·나스닥선물(NQ=F)
- **종목 카드** — 실시간 가격·등락률, 시세 변동 플래시, ±% 표시 임계값(🔔), 🎯 목표가.
- **⚡ 실시간 라인** — 초단위 누적 차트. 묶음 간격(1~600초) 설정.
- **🕔 오늘 흐름 표** — 분 간격 **자유 입력(1~120분)**, 1분봉을 합쳐(resample) 표시.
- **📈 히스토리 차트** — 1개월~1년 일봉, 보조지표.
- **📅 일별 종가 표**.
- **재무 요약** — KIS 기반 PER/PBR/EPS/BPS/시총/52주 고저/거래량회전율 (실시간).
- **🎯 목표가 알림** — 종목별 목표가 도달 시 **카드 강조 + 브라우저 알림 + 사이렌**. localStorage 저장.
- **⚙️ 설정 통합** — 표시 임계값·흐름 간격·실시간 묶음·목표가, 모두 이 브라우저(localStorage)에 저장.
- **세션 인식** — 프리장/본장/에프터장/장마감 배지(KST).

### 디자인 / 브랜딩
- **레이더 브랜드** — 탭 파비콘(레이더+빨강 화살표), 헤더 로고(레이더+위성안테나, radar6), 다크 네이비+골드 테마.
- 알림음 = **사이렌**.

### 인프라 / 배포
- **VPS** (Ubuntu, systemd `stock-watchdog`) — 24시간 상시.
- **Cloudflare Tunnel** — `https://jessystock.com` HTTPS, 포트 개방 불필요.
- **GitHub Pages** — `docs/` 정적 미리보기(무료 소스, 15분, cron 갱신).
- 배포: `git pull && bash deploy/deploy.sh`.

### 알려진 한계
- 텔레그램 알림은 미설정(봇 토큰 + 알림 데몬 필요).
- `http://<VPS IP>:8000` 직접 접속 가능(방화벽 8000 차단 권장).
- 시간외 분봉은 무료 소스에 없음(그 시간대는 ⚡초단위/카드만).
